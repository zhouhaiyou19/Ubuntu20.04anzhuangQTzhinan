   2. **安装 GCC 和 G++**  
      指导你安装 GCC 和 G++ 编译器，确保你的系统具备编译 QT 项目所需的工具。

      3. **下载 QT**

            5. **故障解决**  
               列出了在安装过程中可能遇到的一些常见问题及其解决方法，帮助你顺利完成安装。
